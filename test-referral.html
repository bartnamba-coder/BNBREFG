⚙️ How It Works

  1. URL Processing: App extracts referral code from URL parameter ?ref=CODE
  2. Code Lookup: Converts referral code to Ethereum address using local mapping
  3. Contract Integration: Passes referrer address to contract's buyToken(_amount, _referrer) function
  4. Fallback: Uses zero address if no valid referrer found

🛠️ Technical Implementation

Contract Function Used:

buyToken(uint256 _amount, address _referrer) payable
